# OpenGraph Gen

A tool/library for generating Open Graph metadata for web pages and applications. Open Graph metadata improves content sharing and link previews across social platforms such as Facebook, Twitter, and LinkedIn by providing structured information (e.g., title, description, image, URL) in your HTML.

## Features

- **Automatic Open Graph Tag Generation:** Generate OG tags dynamically for your web pages.
- **Customizable Metadata:** Easily override defaults for title, description, image, and other properties.
- **Multi-platform Support:** Ensures compatibility with Facebook, Twitter Cards, LinkedIn, etc.
- **Easy Integration:** Simple APIs for popular frameworks and static sites.
- **Extensible:** Add additional meta tags as needed.

## Configuration

You can configure the following options:

- `title`: Title of the page/content
- `description`: Short description
- `url`: Canonical URL
- `image`: Image URL for the preview
- `type`: Object type (`website`, `article`, `profile`, etc.)
- `site_name`: (optional) Name of the website
- `locale`: (optional) Language locale (e.g., `en_US`)

## Supported Platforms

- Facebook
- Twitter (Cards)
- LinkedIn
- WhatsApp
- Discord
- Telegram
